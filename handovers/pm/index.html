3) Code & Ops Conventions

  • Diff caps: default ≤1 file / ≤100 lines unless directive states otherwise. W‑002 used ≤3 files / ≤120 lines; W‑002a ≤2 files / ≤100 lines; W‑002b ≤2 files / ≤120 lines.

  • Branch/Tag scheme: feat/v<phase>-<desc>, backup/post-v<phase>-<desc>, tag v<phase>-<summary>.
  • Acceptance bundles (return exactly): tree, required smoke outputs, HEAD SHA (short or full), and any explicit artifacts.
