IX. Mirror Management

PM must ensure:

  1. Worker mirror is public-safe (contains only worker content).

  2. PM mirror is full private record (contains everything).

  3. scripts/publish.sh completes cleanly with no syntax errors.

  4. Each publish includes slug updates and valid hash rewrite.

If any restricted content appears in the worker mirror ‚Üí stop immediately and re-run prune logic.

X. Planner & Oversight Coordination

PM ensures Planner output never leaks to worker mirror.

PM signs off all accepted change proposals before merge.

  2. All ledgers and plans saved.

  3. Publish logs clean.

  4. PM Handover committed and visible in /ledgers/.

  5. Worker and PM mirrors confirmed clean via ls checks.

  6. Optional: rotate PM slug for privacy.
